NPC to Place - Documentation du module

À propos du module

NPC to Place vous permet de transformer un PNJ classique (type npc) en fiche de planète ou de lieu dans l’univers Spatiopolis, avec une fiche visuelle dédiée et immersive.

Fonctionnalités

  • Deux thèmes visuels : Science-Fiction et Médiéval
  • Champs personnalisés pour les planètes et lieux
  • Listes déroulantes éditables depuis le panneau de configuration de Foundry
  • Bouton intégré pour réinitialiser les listes selon la langue active

Notice d’utilisation

  1. Créez un PNJ (type npc) dans Foundry VTT
  2. Ouvrez la fiche et changez la feuille d’affichage (Sheet) dans l’onglet Feuille :
    ➤ Choisissez « NPC to Place – Planète » ou « NPC to Place – Lieu »
  3. Remplissez les champs selon le type de fiche
  4. Vous pouvez éditer les listes déroulantes depuis :
    Configuration du Monde → Modules → NPC to Place
  5. Un bouton « Réinitialiser les paramètres » permet de restaurer les listes selon la langue actuelle

About the Module

NPC to Place lets you convert a standard NPC (type npc) into a planet or place sheet within the Spatiopolis universe, with immersive custom visuals.

Features

  • Two visual styles: Sci-Fi and Medieval
  • Custom sheet fields for planets and places
  • Dropdown lists are editable via the module’s configuration panel
  • Reset button to reload lists based on the current language

User Guide

  1. Create a new NPC actor (type npc) in Foundry
  2. Open the sheet and change the sheet type via the Sheet tab:
    ➤ Select "NPC to Place – Planet" or "NPC to Place – Place"
  3. Fill out the relevant fields
  4. You can edit dropdown lists from:
    World Settings → Modules → NPC to Place
  5. Use the “Reset Settings” button to reload all values based on the active language

Sobre el módulo

NPC to Place te permite transformar un PNJ (tipo npc) en una ficha de planeta o lugar dentro del universo Spatiopolis, con un estilo visual personalizado.

Características

  • Dos estilos visuales: Ciencia ficción y Medieval
  • Campos personalizados para planetas y lugares
  • Listas desplegables editables desde la configuración del módulo
  • Botón para restablecer listas según el idioma activo

Guía de uso

  1. Crea un PNJ (tipo npc) en Foundry
  2. Abre la hoja y cambia el tipo de hoja desde la pestaña Sheet:
    ➤ Elige "NPC to Place – Planeta" o "Lugar"
  3. Rellena los campos necesarios
  4. Las listas desplegables pueden editarse desde:
    Configuración del Mundo → Módulos → NPC to Place
  5. Usa el botón "Restablecer configuración" para actualizar listas según el idioma

Über das Modul

NPC to Place ermöglicht es dir, einen NSC (Typ npc) in ein Planeten- oder Ortsblatt im Spatiopolis-Universum zu verwandeln, mit individuellem visuellen Stil.

Funktionen

  • Zwei visuelle Stile: Science-Fiction und Mittelalterlich
  • Eigene Felder für Planeten und Orte
  • Auswahllisten können in den Moduleinstellungen angepasst werden
  • Schaltfläche zum Zurücksetzen der Listen nach Sprache

Bedienungsanleitung

  1. Erstelle einen NSC (Typ npc) in Foundry
  2. Öffne das Blatt und ändere den Typ über den Reiter Sheet:
    ➤ Wähle "NPC to Place – Planet" oder "Ort"
  3. Fülle die Felder aus
  4. Listen können bearbeitet werden unter:
    Welteneinstellungen → Module → NPC to Place
  5. Verwende den "Einstellungen zurücksetzen"-Button, um Listen entsprechend der Sprache neu zu laden
